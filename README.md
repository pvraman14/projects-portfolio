# Projects Portfolio

A React portfolio showcasing 10 interactive web projects with search, modal and full-page view modes.

## Screenshots

![Portfolio home grid view](/public/screenshots/home.png)

![Project full-page view](/public/screenshots/page-view.png)

## Setup

```bash
npm install
npm start
```

Opens at `http://localhost:3000`.

## Projects

1. **Countdown Timer** - Live countdown to New Year
2. **Drawing App** - Canvas drawing tool with brush controls
3. **GitHub Profiles** - GitHub user search via GitHub API
4. **Movie App** - Movie browser using TMDB API
5. **Notes App** - Notes with markdown and local storage
6. **Password Generator** - Configurable secure password generator
7. **Quiz Project** - Multiple choice quiz with score tracking
8. **Recipe App** - Recipe search using MealDB API
9. **ToDo App** - Task manager with click-to-toggle and right-click-to-delete
10. **Weather App** - Real-time weather via OpenWeather API

## Stack

- React 18, React Router DOM
- Vanilla JS, HTML5 Canvas
- CSS3 (Grid, Flexbox)
